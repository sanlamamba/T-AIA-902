2. tat de l'Art

2.1 Fondements de l'Apprentissage par Renforcement

L'Apprentissage par Renforcement (RL) traite les problmes de prise de dcision squentielle o un agent apprend des politiques optimales par interaction avec l'environnement [Sutton & Barto, 2018]. Le problme de navigation du taxi illustre parfaitement le compromis exploration-exploitation fondamental au RL.

2.2 Apprentissage par Diffrence Temporelle

Q-Learning [Watkins & Dayan, 1992] et SARSA [Rummery & Niranjan, 1994] reprsentent les mthodes fondamentales de diffrence temporelle :

  • Q-Learning : Mthode hors-politique apprenant les valeurs Q(s,a) par mises jour de Bellman
  • SARSA : Mthode sur-politique mettant jour les valeurs Q bases sur la slection d'action relle

2.3 Apprentissage par Renforcement Profond

Les Deep Q-Networks [Mnih et al., 2015] tendent Q-Learning avec approximation de fonction par rseau de neurones et rejeu d'exprience, permettant l'apprentissage dans des espaces d'tats haute dimension.

Annexe A : Code Pseudocode des Algorithmes

A.1 Implmentation Q-Learning

def q_learning(env, episodes, alpha, gamma, epsilon, epsilon_decay, epsilon_min):
    Q = initialiser_table_q(env.observation_space.n, env.action_space.n)
    
    for episode in range(episodes):
        state = env.reset()
        total_reward = 0
        
        while not done:
            # Slection d'action -greedy
            if random.random() < epsilon:
                action = env.action_space.sample()
            else:
                action = np.argmax(Q[state])
            
            next_state, reward, done, _ = env.step(action)
            
            # Mise  jour Q-Learning
            Q[state, action] += alpha * (
                reward + gamma * np.max(Q[next_state]) - Q[state, action]
            )
            
            state = next_state
            total_reward += reward
        
        epsilon = max(epsilon_min, epsilon * epsilon_decay)
    
    return Q

D.1 Graphique de Progrs d'Entranement

def plot_training_progress(training_histories):
    fig, axes = plt.subplots(2, 2, figsize=(15, 10))
    
    for algo, history in training_histories.items():
        # Smooth rewards with moving average
        smoothed_rewards = np.convolve(
            history['rewards'], 
            np.ones(50)/50, 
            mode='valid'
        )
        
        axes[0,0].plot(smoothed_rewards, label=algo)
        axes[0,1].plot(history['steps'], label=algo, alpha=0.7)
    
    axes[0,0].set_title('Rcompenses d\'Entranement')
    axes[0,1].set_title('tapes par pisode')
    plt.legend()
    plt.tight_layout()
    return fig
